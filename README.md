# minecraft-be-health
A plugin for the LiteLoaderBDS platform that adds a scoreboard under players' names to display their health.

## Dependencies  
This plugin requires [LiteLDev/LiteLoaderBDS](https://github.com/LiteLDev/LiteLoaderBDS)

## Description  
1. This plugin uses the scoreboard `syhealth`. If your server is already using this scoreboard, please manually change it to another scoreboard.  
2. The APIs and functions of the latest version of LiteLDev/LiteLoaderBDS may change. Please stay informed.  
3. This plugin updates players' health values every 500 milliseconds. If you think this interval is too short or too long, please adjust it in the code.  
4. Place this plugin in the default plugin directory of LiteLDev/LiteLoaderBDS, usually `./plugins`.
